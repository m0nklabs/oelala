# Grok Imagine UI Reference (Screenshots)

This folder contains UI reference screenshots for the intended dashboard layout and navigation patterns.

## How to add the screenshots

- Put the raw PNG files in `docs/ui-reference/grok-imagine/raw/`.
- Use the filenames listed below exactly.
- Keep them unedited (no crops) so spacing and proportions remain accurate.
